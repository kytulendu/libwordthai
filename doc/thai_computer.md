## ไมโครคอมพิวเตอร์ 8 บิต

คอมพิวเตอร์ 8 บิต ที่รองรับภาษาไทย เท่าที่ทราบก็จะมี

* Apple II หรือรุ่นเลียนแบบ (พ.ศ. 2524)
* NEC PC-8001 (จากภาพในหนังสือเรื่องน่ารู้เกี่ยวกับไมโครคอมพิวเตอร์ พ.ศ. 2524)
* OKI if800 model20 (จัดจำหน่ายโดย Sahaviriya Infotech Computer (SIC) Co.,Ltd. พ.ศ. 2524?)
* OKI if800 model30 (จัดจำหน่ายโดย Sahaviriya Infotech Computer (SIC) Co.,Ltd. พ.ศ. 2524?)
* Apple III (พ.ศ. 2525)
* Osborne 1 (จัดจำหน่ายโดย Computer Union Co.,Ltd. พ.ศ. 2525)
* Fujitsu Micro 8 (จัดจำหน่ายโดย Unicomp International Co.,Ltd. พ.ศ. 2525)
* EPSON QX-10 (จัดจำหน่ายโดย Sahaviriya Infotech Computer (SIC) Co.,Ltd. พ.ศ. 2526)
* MSX (MSX-ไทย โดย Visarn Oucharearn (วิศาล อู่เจริญ) และ Kwi Co.,Ltd. พ.ศ. 2531)

ระบบภาษาไทยของไมโครคอมพิวเตอร์แบบ 8 บิต ส่วนใหญ่บริษัทผู้นำเข้าจะเป็นผู้พัฒนาระบบภาษาไทยเอง ไม่มีมาตรฐานทั้ง API และการเข้ารหัสตัวอักษรไทย จึงทำให้ใน พ.ศ. 2527 มีรหัสตัวอักษรไทยที่แตกต่างกันมากกว่า 20 แบบ  รหัสภาษาไทยที่นิยมใช้เป็นแม่แบบคือ รหัสภาษาไทยของ ม.เกษตรศาสตร์ เรียกอีกย่างว่า รหัสเกษตร (Kaset)  พัฒนาขึ้นโดยห้องปฏิบัติการวิจัยไมโครคอมพิวเตอร์ มหาวิทยาลัยเกษตรศาสตร์ นำโดย ยืน ภู่วรวรรณ เริ่มการพัฒนาบนเครื่องไมโครคอมพิวเตอร์ NEC PC-8001 ในปี พ.ศ. 2524 ซึ่งเป็นส่วนหนึ่งของการพัฒนาโปรแกรม ขวัญใจนักพิมพ์ดีด หรือ Thai Easy Writer/Thai Word Processing ที่ออกรุ่นแรกสำหรับเครื่องคอมพิวเตอร์ Osborne-1 และ Victor 9000 ในปี พ.ศ. 2525

รหัสเกษตรมีอยู่ 3 รุ่นด้วยกัน รุ่นที่นิยมใช้กันคือรุ่นที่ 3 โดยรหัสเกษตรนี้แตกต่างจากรหัส สมอ. (TIS-620) คือ จะไม่มีเลขไทย, ฃ ขวด, ฅ คน, ตัวลือ (ฦ), ลากข้างยาว (ๅ), พินธุ (อฺ), ยามาการ (อ๎), ฟองมัน (๏), อังคารคู่ (๚), โคมุล (๛) และเครื่องหมายบาท (฿)

คอมพิวเตอร์ 8 บิตที่รองรับภาษาไทยที่นิยมใช้กันจะเป็น Apple II รุ่นเลียนแบบ เนื่องจากมีราคาถูก มีซอฟต์แวร์สำเร็จรูปมาก (เกมส์ก็มากด้วย) และสามารถต่อขยายระบบได้ง่าย มีการ์ดขยายระบบมากมายหลายแบบ สามารถใส่การ์ด CPU Z80 เพื่อใช้งานระบบปฏิบัติการ CP/M และโปรแกรมที่รันบน CP/M เช่น WordStar, SuperCalc, dBASE II และ MBASIC ซึ่งเป็นที่นิยมในภาคธุรกิจในสมัยนั้นได้

ระบบภาษาไทยของ Apple II จะมีอยู่ 2 แบบ คือ

* การ์ดจอ 80 คอลัมน์ที่ใส่รอมตัวอักษรไทยลงไป

    ระบบภาษาไทยแบบนี้จะใช้กับระบบปฏิบัติการ CP/M  โดยเครื่อง Apple II จะต้องใส่การ์ด CPU Z80 และการ์ดจอ 80 คอลัมน์ (Vindex Videoterm หรือการ์ดเลียนแบบ) ที่ใส่รอมตัวอักษรไทยลงไป ใช้รหัสภาษาไทยเกษตร (ไม่มีเลขไทย, ตัวอักษรที่ไม่ได้ใช้ เช่น ฃ ขวด, ฅ คน ฯลฯ และจะมีรหัสตัวอักษรสำหรับวรรณยุกต์รวม) และเขียนซอฟต์แวร์ที่รันบน CP/M เพื่อแสดงผลภาษาไทยแบบ 3 ระดับ 8 บรรทัด

* การ์ดภาษาไทย "Thai AppleSoft card"

    ระบบภาษาไทยนี้ใช้ได้กับ AppleSoft BASIC และ Apple DOS  พัฒนาขึ้นโดย ภาณุทัต เตชะเสน จาก บริษัท ยูนิเทค (ประเทศไทย) จำกัด  ตัวการ์ดรุ่น 2.x จะเป็นการ์ดรอม มีชิปรอม 2 ตัว และชิปลอจิคอีก 3 ตัว สามารถแสดงผลภาษาไทย 40 คอลัมน์ 12 บรรทัด แบบ 4 ระดับ โดยใช้ตัวอักษรแบบบิตแมพ สามารถใช้กับ Apple DOS 3.3 และสามารถตั้งชื่อไฟล์เป็นภาษาไทยได้  การ์ดนี้จัดจำหน่ายโดย บริษัท ซีพียู กรุ๊ป จำกัด และ หจก. ไมโครเวย์ ขายในราคา 1,500-1,000 บาท ในปี พ.ศ.2527

โปรแกรมสำเร็จรูปภาษาไทยส่วนใหญ่จะเป็นโปรแกรมที่รันบน CP/M  มีโปรแกรม ขวัญใจนักพิมพ์ดีด (Thai Easy Writer), เวิร์ดรามา, Thai dBASE II (TBASE), ซูเปอร์คาลไทย, MBASIC ภาษาไทย (ไทยเบสิค) และโปรแกรมบัญชีต่างๆ

## ไมโครคอมพิวเตอร์ 16 บิต

คอมพิวเตอร์ 16 บิต ที่รองรับภาษาไทย เท่าที่ทราบก็จะมี

* Victor 9000/ACT Sirius 1 (จัดจำหน่ายโดย Micro World Co., Ltd. พ.ศ. 2525)
* NEC APC (พ.ศ. 2525)
* IBM PC (จัดจำหน่ายโดย Computer Union Co., Ltd. พ.ศ. 2525)
* Sanyo MBC-550 (พ.ศ. 2525)
* Columbia Data Products MPC 1600 (พ.ศ. 2525)
* Olympia People (พ.ศ. 2526)
* TeleVideo TS 1603 (พ.ศ. 2526)
* Texas Instruments Professional Computer (พ.ศ. 2526)
* ACT Apricot (พ.ศ. 2526)
* NEC APC III (พ.ศ. 2527)
* Wang Professional Computer (จัดจำหน่ายโดย บริษัท อินโนเวชั่นคอร์ปอเรชั่น จำกัด พ.ศ. 2528)
* Uniwell (จัดจำหน่ายโดย ไทยวาสโก)
* TAVON PC (ถาวรคอมพิวเตอร์ ไมโครคอมพิวเตอร์รุ่นเทียบเท่า IBM PC ของไทย)

ไมโครคอมพิวเตอร์แบบ 16 บิต จะเป็นแบบ IBM PC Compatible ใช้ระบบปฏิบัติการ MSDOS หรือ IBM PCDOS  ช่วงแรกๆ (พ.ศ. 2525) นั้น เช่นเดียวกับไมโครคอมพิวเตอร์แบบ 8 บิต บริษัทผู้นำเข้าจะเป็นผู้พัฒนาระบบภาษาไทยเอง ไม่มีมาตรฐานทั้ง API และการเข้ารหัสตัวอักษรไทย จึงทำให้ใน พ.ศ. 2527 มีรหัสตัวอักษรไทยที่แตกต่างกันมากกว่า 20 แบบ รหัสภาษาไทยที่นิยมใช้เป็นแม่แบบในช่วงแรกคือ รหัสเกษตร  โดยจะใช้การดัดแปลงตัวอักษรไทยลงไปในรอมอักขระของการ์ดจอ และเขียนซอฟต์แวร์เพื่อแสดงผลภาษาไทยแบบ 3 ระดับ 8 บรรทัด เช่นเดียวกับระบบภาษาไทยของไมโครคอมพิวเตอร์แบบ 8 บิตที่ใช้ระบบปฏิบัติการ CP/M
ในกรณีที่เป็นการ์ดจอโมโนโครม (MDA) หรือการ์ดจอสี (CGA) ผู้ขายบางรายจะบัดกรีสายไฟต่อกับสวิชต์ หรือทำวงจร Soft Switch โดยใช้พอร์ต IO เพื่อเลือกรอมอักขระไทย/อังกฤษ ทำให้เมื่อใช้งานโปรแกรมภาษาอังกฤษ การแสดงผลตัวอักษรตารางจะไม่ผิดเพี้ยน
โปรแกรมที่แสดงผลภาษาไทยแบบ 8 บรรทัด ก็มีโปรแกรม ขวัญใจนักพิมพ์ดีด, ราชวิถี เวิร์ดพีซี 1.x, ราชวิถี เวิร์ดพีซี 2.x แบบ text mode และโปรแกรมบัญชีรุ่นเก่าของบริษัทสหวิริยา

ต่อมามีการดัดแปลงวงจรของการ์ดจอโมโนโครม โดยเพิ่ม VRAM จาก 4KB ให้เป็น 8KB และโปรแกรม CRTC ให้ทำงานแบบ interlace mode แสดงผลตัวอักษร 50 บรรทัด ทำให้สามารถแสดงผลภาษาไทยแบบ 3 ระดับได้ 17 บรรทัด และแบบ 2 ระดับได้ 25 บรรทัด

ต่อมาในปี พ.ศ. 2526 คุณสักรินทร์ "แวน" สุวรรณนุกูล ได้ประดิษฐ์การ์ดจอเฮอร์คิวลิสขึ้นมา การ์ดจอนี้จะเหมือนการ์ดจอโมโนโครมทุกประการ แต่สามารถแสดงผลแบบกราฟฟิก 720 x 348 ได้ ทำให้สามารถทำระบบภาษาไทย 25 บรรทัด โดยแสดงผลด้วยกราฟิกโหมดแบบไม่ต้องดัดแปลงฮาร์ดแวร์แต่อย่างใด และเริ่มมีผู้เขียนโปรแกรมภาษาไทยแบบกราฟฟิก เช่น ออร์คิด เวิร์ดโปรเซสเซอร์ (บริษัท 315 จำกัด พ.ศ. 2529), R.LAB Word v2.0 (พ.ศ. 2531), CU-Writer (มหาวิทยาลัยจุฬาลงกรณ์ พ.ศ. 2532) และไดรเวอร์ภาษาไทยแบบกราฟฟิก เช่น `VTHAI.COM` (วิชัย ปฏิภาภรณ์ พ.ศ. 2532)
สำหรับภาษาไทย 17 หรือ 25 บรรทัดแบบเท็กซ์โหมดกับการ์ดจอเฮอร์คิวลิสนั้น จะดัดแปลงวงจรให้ CRTC เห็น VRAM ในเท็กซ์โหมดจาก 4KB ให้เป็น 8KB และโปรแกรม CRTC ให้ทำงานแบบ interlace mode คล้ายกับการดัดแปลงการ์ดจอโมโนโครม
ท้ายยุคของการ์ดจอโมโนโครมภาษาไทย (การ์ดภาษาไทย) ก็มีการ์ด iRC Monochrome Graphic Card (Thai Standard) ของ บริษัท iRC ออกวางขายในปี พ.ศ. 2531 การ์ดภาษาไทยตัวนี้ใช้ชิปการ์ดจอของ Tseng Labs ET1000 ดัดแปลงวงจรมาจากการ์ดจอ DFI MG-150

ต่อมาในปี พ.ศ. 2527 ก็มีการ์ดจอ EGA ออกวางขาย  ในปี พ.ศ. 2530 ทาง IBM ก็ออกคอมพิวเตอร์ IBM PS/2 ที่มีการ์ดจอ VGA ออนบอร์ดเป็นมาตรฐานของเครื่องรุ่นนี้  ซึ่งการ์ดจอ EGA และ VGA สามารถทำ Soft Font ได้ ทำให้สามารถทำระบบภาษาไทย 25 บรรทัดด้วยซอฟต์แวร์อย่างเดียวโดยที่ไม่ต้องแก้ไขฮาร์ดแวร์ได้เลย  ระบบภาษาไทย EGA และ VGA จะเป็นไดรเวอร์ TSR มักใช้กับฮาร์ดล็อคหรือดองเกิลที่ต้องเสียบกับพอร์ตพริ๊นเตอร์เมื่อรันไดรเวอร์เพื่อป้องกันการก๊อปปี้

ไดรเวอร์ภาษาไทย EGA/VGA ก็จะมี

* AXTHAI (Actinix Co.,Ltd.)
* iRC VGA Pro aka. ThaiPro (International Research Corporation Co.,Ltd. (iRC))
* MicroWiz Dyna Key, MicroWiz XTRA
* Olympia Thai driver (EGA) (DOS 36 Co., Ltd.)
* Powersoft Thai Professional
* RLAB Thai Driver (R.LAB & Consultant Co., Ltd.)
* Thai Driver for IBM PS2 (Computer Union Co., Ltd.)
* Thai Driver for IBM PS2 (Metro System Co.,Ltd., พัฒนาจาก R.LAB Thai Driver)
* Thai System Manager (TSM) for DOS (ThaiSoft Co.,Ltd.)
* THAILS ของ MS-DOS 6.22 Thai Edition (พัฒนาโดย บริษัท iRC)
* TSCC Plus VGA EGA (Summit Computer Co.,Ltd.)
* Venus Thai Driver (Computer Union Co., Ltd.)
* VTHAI (วิชัย ปฏิภาภรณ์)

นอกจากนี้ยังมีการ์ดภาษาไทยสำหรับ VGA ของ บริษัท R&D (บริษัทนี้ทำการ์ดป้องกันไวรัส AVC-2000/AVC-4000/AVC-5000) ชื่อ R&D Thai On Board VGA ซึ่งเป็นการ์ด ISA มีรอมบรรจุ BIOS ภาษาไทย เมื่อเสียบลงเครื่อง เปิดเครื่องมาแล้วจะสามารถใช้งานภาษาไทยได้ทันที โดยไม่ต้องโหลดไดรเวอร์ภาษาไทย

สำหรับมาตรฐานระบบภาษาไทยนั้น ในช่วงแรก ปี พ.ศ. 2526 มีการพยายามจัดทำมาตรฐานการเข้ารหัสภาษาไทยโดย ยืน ภู่วรวรรณ ในบทความ "ถึงเวลาที่จะหารหัสมาตรฐานภาษาไทยแล้วหรือยัง" นิตยสารไมโครคอมพิวเตอร์ ฉบับที่ 4  ต่อมาในปี พ.ศ. 2529

## Microsoft Windows ภาษาไทย

ระบบภาษาไทยสำหรับ Microsoft Windows เริ่มมีการศึกษาพัฒนาโดย ภาณุทัต เตชะเสน จากบริษัท 315 จำกัด ตั้งแต่ยุค Windows 1.0 จนกระทั่ง Windows 2.0 ออกวางขาย จึงได้พัฒนาโปรแกรม tofWrite (Three-One-Five) ซึ่งเป็นโปรแกรมเวิร์ดโปรเซสเซอร์ภาษาไทยบน Microsoft Windows โปรแกรมแรก โปรแกรมนี้เลียนแบบมาจากโปรแกรม Microsoft Write วางขายในราคา 15,000 บาท
ต่อมาได้มีการพัฒนาโปรแกรม tofOrchid-2 ซึ่งเป็นโปรแกรม Desktop Publishing ภาษาไทยโปรแกรมแรกบน Microsoft Windows โปรแกรมนี้เลียนแบบมาจากโปรแกรม Aldus PageMaker วางขายในปี พ.ศ. 2529 ในราคา 35,000 บาท
ต่อมา Windows 3.0 ได้ออกวางขายในเวลาเพียง 4 เดือน ได้ออก ThaiWin 1.01 (พ.ศ. 2534) ซึ่งระบบภาษาไทยสำหรับ Microsoft Windows ตัวแรก

ส่วน Windows Thai Edition รุ่นแรกที่ออกมาคือ Windows 3.0a Thai Edition พัฒนาโดย Microsoft, International Research Corporation (iRC) และ MicroWiz จัดจำหน่ายโดย The Value Systems (พ.ศ. 2534)

ต่อมา Windows 3.1 Thai Edition (พ.ศ. 2536) พัฒนาโดย Microsoft และ The ValueSoft Vision Co.,Ltd.

ถึงแม้ว่าจะมี Windows 3.1 Thai Edition แล้วก็ตาม ก็มีผู้พัฒนาระบบภาษาไทยสำหรับ Windows 3.1 อยู่ โดยจะมี หน้าต่างแบบไทยไทย (Archanetic Thai System), Thai System Manager for Windows (TSMW) ของ บริษัท ThaiSoft และ ThaiWin 2.1 หรือ 3.1 ของ บริษัท 315

ส่วน Windows 95 แม้จะมี Windows 95 Thai Edition ก็มีผู้พัฒนาระบบภาษาไทยอยู่ คือ ThaiMaster 95 ของบริษัท Solution Master ซึ่งบริษัทนี้ทำระบบภาษาไทยของ Windows NT 4.0 ด้วย (ThaiMaster 1.0 for Windows NT 4.0)

Microsoft Windows Thai Edition ที่พัฒนาโดย Microsoft ได้แก่

* Windows 3.0a Thai Edition
* Windows 3.1 Thai Edition
* Windows 3.11 Thai Edition
* Windows 95 Thai Edition
* Windows 98 Thai Edition
* Windows 98 SE Thai Edition
* Windows ME Thai Edition
* Windows NT 4.0 Thai Enabled

สำหรับ Windows 2000 ขึ้นไปนั้น ไม่มีรุ่นภาษาไทยเฉพาะออกมา จะสามารถใช้งานภาษาไทยได้โดยไม่ต้องติดตั้งระบบภาษาไทยเพิ่มแต่อย่างใด

นอกจากนี้ยังมีโปรแกรมที่ใช้งานคู่กับโปรแกรม Desktop Publishing บน Windows คือ iRC ZeWrite, PageMaster 97 ของบริษัท Solution Master และ Kokai 2000 for Windows ของ Prepress Solution Co.,Ltd.

## Apple Macintosh ภาษาไทย

ระบบภาษาไทยบนเครื่องคอมพิวเตอร์แมคอินทอช สำหรับ System 6 และ System 7 นั้น จะเป็นโปรแกรม ThaiTalk พัฒนาโดยสหวิริยา ผู้นำเข้าเครื่องแมคอินทอชในสมัยนั้น

ต่อมาจะเป็น Thai Language Toolkit (TLT) สำหรับ MacOS 8 และ MacOS 9 พัฒนาโดย Macintosh Center โปรแกรม TLT นี้สามารถใช้งานกับ MacOS 7.5 ได้ โดยต้องมีส่วนขยาย CFM-68K 4.0 และ Appearance SDK 1.0.4

สำหรับ Mac OS X นั้น สามารถใช้งานภาษาไทยได้โดยไม่ต้องติดตั้งระบบภาษาไทยเพิ่มแต่อย่างใด

นอกจากนี้ยังมีโปรแกรมที่ใช้งานคู่กับระบบภาษาไทย TLT สำหรับโปรแกรม Desktop Publishing คือโปรแกรม Thai Support Plug-in โดย Macintosh Center Co.,Ltd. และ Kokai 2000 โดย Prepress Solution Co.,Ltd.  โดยโปรแกรมดังกล่าวใช้งานตัดคำภาษาไทย และแก้สระจม/ลอยในโปรแกรม PageMaker, QuarkXPress, InDesign, Photoshop, Illustrator และ Macromedia FreeHand เมื่อใช้งานกับฟอนต์ที่รองรับ

## ภาษาไทยสำหรับระบบปฏิบัติการอื่นๆ

* ภาษาไทยสำหรับระบบปฏิบัติการ GEM ที่ใช้สำหรับรันโปรแกรม Ventura
* ภาษาไทยสำหรับระบบปฏิบัติการ OS/2
* ภาษาไทยสำหรับระบบปฏิบัติการ UNIX (เช่น SGI IRIX, SUN)
* ภาษาไทยสำหรับระบบปฏิบัติการ Linux (โครงการ Linux TLE)
* ภาษาไทยสำหรับระบบปฏิบัติการ Windows CE และ Windows Mobile (ThaiWinCE)
